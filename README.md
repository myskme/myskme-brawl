# MYSKME 大乱斗

> 《狼先生与他的学生们》世界观 · 课堂积分对战工具（iPad / 投屏版）
> 以"三国课堂军师争霸积分器"的玩法引擎为基底，整体换皮为 MYSKME 世界观。

## 使用方式

直接双击 `index.html` 即可使用。**单文件、零依赖、零外部图片、可离线**——所有美术都是程序化（SVG/CSS 字形徽章），不需要 `assets/` 文件夹。

适合 1–6 位玩家、课堂投屏与 iPad。

## 这是什么

光明阵营的学员（猫女琳、雪豹睿、鹿女海、白狐月……）在狼先生（Mr. Wang）的课堂上互相争霸，
也可以切换"同盟远征"，组队讨伐黑域·蛇女王叶王。答题加分驱动一切。

### 玩法速览

1. **随机点名 / 加分**：点每位学员的 +1 / +2，决定排名。
2. **星尘经营**：答题获得星尘 → 布据点收租、升修行、招募战技、抢占星力。
3. **职业技能**：六大职业各有被动 + 3 回合冷却技能。
4. **羁绊卡**：答题可能掉卡，最多留 3 张。
5. **黑域远征（塔防）**：见下。

## ⚔️ 黑域远征 · 回合制 JRPG 战团（v3 重做）

黑域远征整体重做为**梦幻西游 / 女神异闻录式的纯回合制群战**，一键好上手：黑域**成群**来袭（每战 2–5 只同屏），**答题即出手**——点选集火目标，攒满技力放必杀，攻防之间守住光明壁垒。

- **敌人花样**：每只敌人带行动条（ATB），并可能拥有 **影分身 / 复活 / 分裂 / 嗜血 / 疾驰 / 凝盾 / 扰乱 / 狂暴 / 召唤** 等技能；首领多技能、残血狂暴。
- **角色主被动 + 必杀**：六职业各有被动（觉醒暴击 / 护守减伤 / 越界连击 / 调和回血 / 决斗增伤 / 观星充能）与 **技力满后一键必杀**（觉醒斩 / 守护结界 / 急行突袭 / 星尘治愈 / 决斗宣言 / 星象预言），搭配策略性强。
- **经验养成**：击败敌人全队涨经验、升级、壁垒上限提升、暴击率随等级提升。
- **黑域地牢**：每战之间在 2–3 条岔路里抉择——**黑域宝箱（开盒赌一把：惊喜 or 陷阱，越深越大）/ 精英险路 / 星语营地（回血充能）/ 常规战团**。
- **连锁热度（HEAT）→ 暴击**；**光明审判（OVERDRIVE）**蓄满一键全屏清场；**无伤战**奖励；焦点集火（点敌人卡或头像，不锁定则自动打最前排）。
- 壁垒归零则本局结束，可输入管理员密码续命。

## 🤝 同盟军备（合作模式 · v3 新增）

多人合作（同盟远征）下，把**池化的团队总分**大量转换为远征战力：**强化壁垒 / 全队充能 / 星尘补给 / 复苏徽记**，让合作攒下的分数真正变成实力。

## 🎨 SVG 美术（v3 新增）

敌人与角色全部用**程序化 SVG 剪影**渲染（狼 / 牛 / 隼 / 鹿 / 鲨 / 鸮 + 蛇 / 鸦 / 虎 / 狞猫 / 蛇女王 / 藤蔓 / 幽火 / 召唤笼…），按阵营色实时着色，无 emoji、零外部图片。

## 🏆 结算 · 炫耀卡（v2 新增）

**每个模式都能一键结算 → 生成竖版炫耀战报卡（1080×1920，带二维码）→ 保存到本地**：

- **全员大乱斗 → 封王战报**：冠军 + 王冠 + 称号 + **全员战绩榜**（每个孩子都上榜）+ 本节 MVP + **逆袭之星**（逆袭最多者）。
- **同盟远征 → 同盟战报**：团队总分 + 段位印章（微光 / 破晓 / 曜日 / 圣堂）+ **全员同等署名**功勋块。
- **黑域远征 → 守壁战报**：波次 / 击退 / 完美波 / 壁垒残存 / 守壁英雄 + 高光成就。

计分台「🏆 结算炫耀卡」按钮即可结算；「历史战报」可随时回看本机最近三张卡。

## 视觉风格 · 黑曜烫金 Obsidian Foil（v2 升级）

由「黑金编年史」升级为更高级的 **黑曜烫金**：

- 退掉塑料感荧光黄，改用**三段烫金渐变**（#8a6a2e → #c9a86a → #f4e3b0），标题与分数都是「烫金字」。
- 基底加深为黑曜石（#070608），暖（光明）冷（黑域）两极氛围光。
- 稀有度宝石化（普通灰 / 稀有蓝 / 史诗紫 / 传说金 / 神话朱）、面板金属描边、等宽数字。
- 全程序化字形徽章 + canvas 渲染，零外部图片，离线可用。
- 大量打击感：浮动伤害/得分数字、受击闪光、破壁红闪+震屏、完美波金环、封王印章砸入。
